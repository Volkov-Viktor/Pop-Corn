\# PopCorn



Аркадная игра.



!\[Начальный экран игры](screenshots/start.png "Скриншот начального экрана")





\## Особенности



\- Взрывающиеся кирпичи (Explosive Bricks)

\- Лазерные платформы (Laser Platforms)

\- Монстры, мешающие игроку (Monsters)

\- Система сбора букв для завершения уровня

\- Расширяющаяся платформа (Platform Expansion)

\- Механика «клейкой» платформы (Glue Platform)



!\[Кадр игры](screenshots/level1.png "Скриншот первого уровня")





\## Лицензия



PopCorn - некоммерческий учебный проект по курсу А.Семенко "Базовый курс программирования на С++".

С изменениями и дополнениями.



Визуальная и игромеханическая основа - аркада Popcorn 1988 от разработчика Lacral Software.

Оригинальные код и файлы изображений при написании данного проекта не использовались.
